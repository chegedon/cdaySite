Admissions Assistant

AI-Enhanced Chatbot for Graduate Admissions

Project Overview

Admissions Assistant is a proof-of-concept project designed to enhance the visitor experience of Kennesaw State University’s Graduate Admissions website by integrating an intelligent sample chatbot. Its purpose is to guide users to the information they need quickly and efficiently—reducing unnecessary manual intervention from admissions personnel.

The KSU Graduate Admissions website contains extensive information about programs, application steps, deadlines, and eligibility requirements. However, navigating this content can be challenging for prospective students. This often results in user frustration and increased dependency on admissions counselors to answer repetitive questions. These interruptions divert counselor time away from more complex advising needs, creating service delays and administrative strain.

This project addresses those challenges by introducing a chatbot capable of retrieving and presenting the website information in a more intuitive, conversational format—improving user satisfaction and reducing workflow overhead for admissions staff.

Project Objectives


    ✔ Develop a chatbot capable of autonomously updating its knowledge base from the KSU website.

    ✔ Ensure the chatbot provides accurate website references for user queries.

    ✔ Build a user-friendly interface allowing admissions staff to correct chatbot responses.

    ✔ Demonstrate feasibility by integrating the chatbot into a static clone of the KSU Graduate Admissions website.

Scope of Work

The team will develop the Admissions Assistant chatbot on a static clone of KSU’s existing Graduate Admissions website. This prototype will demonstrate the viability of the solution and highlight how AI-driven conversational systems can streamline information access for prospective students.
